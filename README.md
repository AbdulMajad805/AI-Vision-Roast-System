# 🤖 AI Vision Roast System 😂🔥

An AI-powered web application that generates funny and sarcastic roasts based on user images using Computer Vision and Natural Language Processing.

---

## 🚀 Features

- 📸 Upload image or take photo using camera  
- 🧠 AI-based image captioning (BLIP model)  
- 😂 Dynamic roast generation using LLM (Groq API)  
- ⚡ Real-time processing  
- 🎨 Clean and interactive UI (Streamlit)  
- 🌐 Fully deployed on Streamlit Cloud  

---

## 🧠 How It Works

1. User uploads an image or captures a photo  
2. Image is processed using a vision model (BLIP)  
3. The model generates a caption describing the image  
4. The caption is sent to a language model via Groq API  
5. The system generates a humorous roast  
6. The result is displayed on the UI  

---
